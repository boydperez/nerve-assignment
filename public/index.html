Nerve Assignment API Docs.

Developed by Boyden Pires

Get postman for testing these APIs https://www.postman.com/

API ENDPOINTS

Create record

POST   /api/v1/records
{
    “srNo”: <Serial number>,
    “name”: <Name>
}

Get record

GET   /api/v1/records/<sr.no>

Get records

GET    /api/v1/records

Update record

PATCH   /api/v1/records/<sr.no>
{
    “name”: <Name>
}

Delete record

DELETE   /api/v1/records/<sr.no>
